Projections.jl

Projections.jl

Projections.jl is a Julia package that collects commonly used projection operators. It is useful for optimization algorithms such as projected gradient descent.

Given a point $y \in \mathbb{R}^n$ and a set $S \subset \mathbb{R}^n$, the projection of $y$ onto $S$ is defined as $ \text{proj}S(y) = \arg \min{x \in S} \|y - x\|2^2. $ If $S$ is a closed convex set, then \text{proj}S(y)$ is unique.

Basic usage

This package provides two functions for a set $s$,

project!(s, v, y)

overwrites v by projection of y to set s. 

project(s, y)

simply returns projection of y to set s.

Projection operators

using Projections

Ball

Projection of $y$ onto the closed ball $S = \{x \in \mathbb{R}^n: \|x - c\|_2 \le r\}$ is $ \text{proj}S(y)i = \begin{cases} c + r \frac{y - c}{\|y - c\|2} & \|y - c\|2 > r \ y & \|y - c\|_2 \le r \end{cases}. $

y = [-2.5, -1.5, -0.5, 0.0, 0.5, 1.5, 2.5]
# norm is less than 5
sqrt(sum(abs2, y))

Box

Projection of $y$ onto a closed box $S = [a_1, b_1] \times \cdots \times [a_n, b_n]$ is $ \text{proj}S(y)i = \begin{cases} ai & yi < ai \ yi & yi \in [ai, bi] \ bi & yi > bi \end{cases}. $

y = [-2.5, -1.5, -0.5, 0.0, 0.5, 1.5, 2.5]
a = fill(-1.0, 7) # vector of all -1
b = fill( 1.0, 7) # vector of all  1
project(Box(a, b), y)

Contributing

You are welcome to contribute to the Projections.jl package. To add a projection operator, e.g., projection to the affine set, to the package, follow these steps:

  1. Fork the Projections.jl package.
  2. Add affine.jl to the src folder that implements the type Affine, the function project!(s::Affine, v, y), and the function project(s, y).
  3. Add a line include("affine.jl") to the /src/Projections.jl file.
  4. Add test file affine_test.jl to the test folder.
  5. Add a line include("affine_test.jl") to the /test/runtests.jl file.
  6. Make sure that Pkg.test("Projections") successfully runs.
  7. Add documentation for the new projection operator to the Projection operators section of Jupyter notebook /docs/Projections.jl.
  8. Issue a pull request.
